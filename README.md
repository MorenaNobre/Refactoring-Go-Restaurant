## Sobre o Projeto

<br>

<p>Esse é um desafio complementar do Ignite para fixação de conhecimento.</p>

<p>O desafio consiste em refatorar uma aplicação já iniciada. O objetivo é realizar dois processos de migração: de Javascript para Typescript e de Class Components para Function Components.</p>

<p>Nesse desafio foram utilizados os seguintes conceitos:</a>

* Fake API com JSON Server;
* Axios
